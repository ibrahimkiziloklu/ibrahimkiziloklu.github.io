Five-Bar Parallel Arm Robot

A precision-focused parallel robot for biomedical applications

Project Overview

I designed and developed the Five-Bar Parallel Arm Robot at the Biomechanics Lab at Aalborg University. This parallel robot was specifically engineered for biomedical applications requiring high precision and optimized kinematics.

Unlike serial robots commonly used in industry, I chose a parallel robot architecture for its unique advantages in terms of rigidity, speed, and precision. My design of the five-bar linkage mechanism provides an optimal balance between workspace size and mechanical advantage, making it ideal for rehabilitation applications and biomedical research.

Mechanical Design

My work involved comprehensive research on motion dynamics and the implementation of nonlinear control systems, resulting in a high-performance robotic platform that is now deployed in various biomedical settings. I developed a control system using ROS (Robot Operating System) with custom C++ nodes for real-time control and Python for higher-level control and user interface.
