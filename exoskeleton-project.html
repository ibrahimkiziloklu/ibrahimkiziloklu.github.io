Exoskeleton for Rehabilitation

Assessing human lower limb mechanics for enhanced sports injury rehabilitation

Project Overview

The Exoskeleton for Assessing Human Lower Limb Mechanics was developed at the Center for Rehabilitation Robotics in collaboration with the Biomechanics Research Group of Aalborg University. This project focused on creating a specialized exoskeleton prototype targeting the study of hamstring muscle mechanics to enhance sports injury rehabilitation.

Hamstring injuries are among the most common and recurrent issues in sports, particularly in activities involving sprinting and jumping. Traditional rehabilitation approaches often lack precise quantitative feedback on muscle mechanics during therapy exercises. This exoskeleton was designed to address this gap by providing accurate measurement and controlled loading capabilities.

As part of my role at the Biomechanics Lab, I revised and upgraded the design of this lower body exoskeleton and prepared necessary exercise patterns for rehabilitation protocols. The project combined principles from biomechanics, robotics, and rehabilitation science to create an effective tool for both research and clinical applications.

Technical Specifications

Feature Specification
Degrees of Freedom 1 active (knee joint), 2 passive (hip and ankle)
Actuation Brushless DC motor with gear reduction
Torque Capacity Peak: 80 Nm, Continuous: 40 Nm
Angular Range 0° - 120° (knee flexion/extension)
Sensors Encoder, force/torque sensor, IMUs (3 units)
Control System ROS2-based real-time controller (1 kHz)
Weight 3.5 kg (wearable components)
Adjustability Accommodates users 160-190 cm height
Operation Modes Passive, assistive, resistive, and assessment

Mechanical Design

The exoskeleton was designed with a focus on ergonomics, adjustability, and measurement precision. Key mechanical design features include:

Exploded view of the exoskeleton's knee joint mechanism showing key components

Anatomically Aligned Joint

The knee joint was designed to align with the natural human knee axis, with self-aligning capabilities to accommodate individual anatomical variations and prevent discomfort during movement.

Modular Frame Structure

Adjustable thigh and shank frames with quick-release mechanisms to fit different body sizes. The aluminum construction balances weight, strength, and rigidity requirements.

Series Elastic Actuation

A custom-designed series elastic element between the motor and joint provides compliant force control, impact absorption, and accurate torque measurement capabilities.

Comfortable Interface

Ergonomic cuffs with pressure distribution padding and adjustable straps ensure comfortable wear during extended rehabilitation sessions. Special attention was paid to prevent pressure points and skin irritation.

The mechanical design went through multiple iterations based on user feedback and biomechanical testing. FEA (Finite Element Analysis) was used to optimize structural components for strength while minimizing weight.

Control System Architecture

One of the major contributions to this project was the development of a modular ROS2-based control system for real-time data acquisition and user-friendly operation. The control architecture consists of several interconnected components:

The control system implements several control modes to support different rehabilitation scenarios:

Zero-Force Mode

The exoskeleton compensates for its own weight and friction, allowing the user to move freely without resistance. This mode is used for initial assessments and range of motion exercises.

Assistive Mode

Provides variable support to the user's movement, particularly useful for patients with limited strength. The assistance level can be adjusted based on rehabilitation progress.

Resistive Mode

Applies controlled resistance to strengthen muscles. The resistance profile can be customized (constant, progressive, or position-dependent) to target specific aspects of muscle function.

Assessment Mode

Records precise measurements of joint angles, forces, and movement patterns for quantitative evaluation of rehabilitation progress.

The control system was implemented using ROS2, which offered advantages in terms of modularity, communications infrastructure, and integration with other research tools. Here's a simplified example of the primary control node:

// ROS2 Exoskeleton Control Node (Simplified)
#include "rclcpp/rclcpp.hpp"
#include "sensor_msgs/msg/joint_state.hpp"
#include "std_msgs/msg/float64.hpp"
#include "geometry_msgs/msg/wrench.hpp"

class ExoskeletonController : public rclcpp::Node
{
public:
  ExoskeletonController() : Node("exoskeleton_controller")
  {
    // Initialize publishers and subscribers
    joint_state_pub_ = this->create_publisher(
      "joint_states", 10);
    
    torque_command_pub_ = this->create_publisher(
      "torque_command", 10);
    
    force_sensor_sub_ = this->create_subscription(
      "force_sensor", 10, 
      std::bind(&ExoskeletonController::forceSensorCallback, this, std::placeholders::_1));
    
    // Create control loop timer (1 kHz)
    timer_ = this->create_wall_timer(
      std::chrono::milliseconds(1),
      std::bind(&ExoskeletonController::controlLoop, this));
    
    // Initialize control parameters
    control_mode_ = ZERO_FORCE_MODE;
    desired_position_ = 0.0;
    desired_torque_ = 0.0;
    
    RCLCPP_INFO(this->get_logger(), "Exoskeleton controller initialized");
  }

private:
  void controlLoop()
  {
    // Read sensor data
    updateSensorData();
    
    // Calculate control output based on mode
    double torque_output = 0.0;
    
    switch(control_mode_) {
      case ZERO_FORCE_MODE:
        torque_output = calculateCompensationTorque();
        break;
      case ASSISTIVE_MODE:
        torque_output = calculateAssistiveTorque();
        break;
      case RESISTIVE_MODE:
        torque_output = calculateResistiveTorque();
        break;
      case ASSESSMENT_MODE:
        torque_output = calculateAssessmentTorque();
        break;
    }
    
    // Send command to motor driver
    sendTorqueCommand(torque_output);
    
    // Publish state information
    publishJointState();
  }
  
  // Other methods and variables...
};

Rehabilitation Applications

The exoskeleton was designed to support a range of hamstring rehabilitation protocols, each addressing specific aspects of the recovery process:

Isometric Strengthening

The exoskeleton holds the knee at specific angles while the patient performs isometric contractions. The system measures the generated force, providing quantitative feedback on strength improvements over time.

Eccentric Training

Controlled eccentric loading (lengthening contractions) is crucial for hamstring rehabilitation. The exoskeleton provides precise resistance during the eccentric phase of movement, with safety mechanisms to prevent overexertion.

Range-of-Motion Exercises

Gradual increase in movement range with adjustable assistance/resistance. The system tracks progress and adapts the difficulty based on patient performance.

Functional Movement Patterns

Sport-specific movement patterns can be programmed and practiced with controlled loading to prepare for return to sports activities.

Each rehabilitation protocol was developed in collaboration with sports physiotherapists and based on current evidence in hamstring injury rehabilitation. The exercise patterns were programmed into the control system and could be customized for individual patients.

The testing phase validated that the system's capabilities met or exceeded the design specifications. The exoskeleton demonstrated reliable performance under both loaded and unloaded conditions, confirming its suitability for the intended rehabilitation applications.

Research Outcomes

The exoskeleton project yielded several valuable outcomes for the field of rehabilitation robotics:

  • Development of a new methodological approach for quantitative assessment of hamstring function during rehabilitation
  • Creation of an open-source ROS2-based control framework that can be adapted for other rehabilitation robotics projects
  • Collection of normative data on hamstring mechanics during various exercise patterns
  • Establishment of design guidelines for ergonomic wearable lower-limb rehabilitation devices
  • Validation of series elastic actuation as an effective approach for safe human-robot interaction in rehabilitation

The project findings are being incorporated into ongoing research at the Center for Rehabilitation Robotics and have informed the development of next-generation rehabilitation devices.

Future Work

Building on the success of the current prototype, several directions for future development have been identified:

Multi-Joint Integration

Expanding the system to include coordinated control of hip, knee, and ankle joints for more comprehensive lower-limb rehabilitation.

Adaptive Control Algorithms

Implementing machine learning approaches to optimize assistance/resistance based on individual patient progress and movement patterns.

Telerehabilitation Interface

Developing a remote monitoring and guidance system to support home-based rehabilitation with the exoskeleton under therapist supervision.

Integrated Biofeedback

Adding EMG sensors and real-time biofeedback to provide patients with immediate information about muscle activation patterns during exercises.

These enhancements aim to further improve the effectiveness of the system in clinical settings and expand its application to a broader range of rehabilitation scenarios.
